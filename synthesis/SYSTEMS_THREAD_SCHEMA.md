# Systems Theory Thread Schema

Live artifact: [agi-research/findings/SYSTEMS_THEORY_THREAD.md](https://github.com/mroncka/agi-research/blob/main/findings/SYSTEMS_THEORY_THREAD.md)

## Thread Structure

The systems theory thread is a **running correlation** — each run adds observations that connect the surface-level events (Track A–D findings) to underlying systemic patterns.

## Correlation Entry Schema

```markdown
### [Run NNN] — [Phenomenon Name]
- **Signal Source**: [finding from Track A/B/C/D]
- **Systems Pattern**: [emergence | phase_transition | power_law | self_organization | cascade | attractor | bifurcation]
- **Mechanism**: [proposed causal mechanism]
- **Evidence Grade**: [CONFIRMED | INFERRED | HYPOTHETICAL]
- **Quantification**: [if applicable: α=X, Cohen's d=Y, n=Z]
- **Prediction**: [what this pattern implies for next 30–90 days]
- **Connects To**: [prior run / prior phenomenon]
```
