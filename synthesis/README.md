# Synthesis Layer

Raw research becomes structured knowledge through a three-stage synthesis process:

1. **Delta Extraction** — identify net-new signals from raw research output
2. **Triple Substrate Update** — route each delta to KG + Tools Registry + Systems Thread simultaneously
3. **Report Generation** — produce the human-readable run report with citations

Synthesis quality is the bottleneck. A well-executed synthesis run produces 3–8 genuine deltas, cleanly typed and positioned in all three substrates.
