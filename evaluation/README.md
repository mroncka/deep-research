# Evaluation

Research quality degrades silently. The evaluation layer exists to catch:

1. **Signal dilution** — runs that produce noise instead of signal
2. **Source concentration** — over-reliance on a single source type
3. **Thesis attachment** — filtering out disconfirming evidence
4. **Query staleness** — queries that no longer surface fresh material
5. **Thesis drift** — gradual, unnoticed shift in research framing

Review cadence: weekly (manual) + flagged automatically when a run scores < 3 genuine deltas.
