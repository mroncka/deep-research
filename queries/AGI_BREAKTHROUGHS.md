# Query Bank: AGI Breakthroughs

## Domain
Capability frontier — what can AI systems do now that they couldn't 30–90 days ago?

---

## Primary Queries

```
AGI agentic systems latest breakthroughs [current month year]
```
```
AI benchmark new record state of the art [current month year]
```
```
LLM emergent capabilities new behaviors [current month year]
```
```
GPT-5 Claude Gemini capability comparison [current month year]
```
```
AI surpasses human performance new domain [current month year]
```

## Follow-up Probes
```
[MODEL NAME] technical report evaluation methodology
```
```
What benchmarks does [MODEL NAME] fail that previous models passed?
```
```
Reproducibility concerns [MODEL NAME] [BENCHMARK NAME]
```

## Contradiction Queries
```
AGI timeline criticism evidence against [current month year]
```
```
LLM capability overstated benchmarks gaming [current month year]
```

## Structured Output Target

For each finding, extract:
- Model name + version
- Benchmark + score (absolute, not relative)
- Human baseline comparison (if available)
- Date of publication/announcement
- Primary source URL
- Credibility tier (Tier 1: peer-reviewed | Tier 2: official announcement | Tier 3: community)
